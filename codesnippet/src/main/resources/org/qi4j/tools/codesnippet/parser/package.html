Contains classes related to parsing java source files for extraction of special comments containing xml.

Example

Input and output of the JavaXmlCommentsReader class.
Java source as input
public class Sample
{
    // $<snippet id="first">
    int /* $<b>$ */first/* $</b>$ */= 0;
    // $</snippet>
}
will result in
XML as output
<?xml version="1.0"?>
<java>
<![CDATA[public class Sample
{
    // $<snippet id="first">
    int ]]><b><![CDATA[first]]></b><![CDATA[= 0;
    // $</snippet>
}]]>
</java>
